# getAir Home Assistant Integration

Control your **getAir ComfortControl Pro BT** ventilation system directly from Home Assistant.

## What you get

- 🌀 **Fan entity** with speed control (0.5–4.0) and ventilation modes
- 🌡️ **Sensors** for indoor temperature, humidity and air quality
- 🎛️ **Custom dashboard cards** for a polished control panel
- 🔐 **Automatic token refresh** — no manual re-authentication needed

## Prerequisites

- A getAir account linked to your device via the getAir app
- API access enabled by getAir support — email `info@getair.eu` to request it

## Supported Devices

- getAir ComfortControl Pro BT
- getAir ComfortControl Pro

## Setup

After installation, go to **Settings → Integrations → Add Integration** and search for **getAir**. Enter your getAir account credentials and you're done!
